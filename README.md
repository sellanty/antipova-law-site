# Сайт адвоката Антиповой Елены Сергеевны

Одностраничный сайт-визитка на Next.js + TypeScript + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

После запуска открыть:

```txt
http://localhost:3000
```

## Что обновлено в этой версии

- Адрес обновлен: Липецк, Студеновская улица, 19, помещение 1.
- Добавлен регистрационный номер: 48/506 в реестре адвокатов Липецкой области.
- Добавлен блок опыта: 26 лет юридического стажа, 17 лет адвокатской деятельности, 9 лет в судебной системе.
- Добавлен визуальный символ Фемиды.

## Где менять данные

Основные данные:

```txt
lib/site-config.ts
```

Тексты, категории, отзывы, FAQ:

```txt
lib/content.ts
```

Секции сайта:

```txt
components/sections/
```

Фотографии:

```txt
public/images/
```

## Карта

Ключ Яндекс.Карт берется из переменной окружения:

```env
YANDEX_MAPS_API_KEY=...
```

Локально он лежит в `.env.local`. При публикации сайта лучше добавить ключ в переменные окружения хостинга и ограничить его по домену.
